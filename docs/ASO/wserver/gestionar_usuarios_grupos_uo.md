# Gestionar cuentas de usuario, de equipos, grupos y unidades organizativas

## Creación de un nuevo usuario
Para crear un nuevo usuario: Herramientas > Usuarios y equipos de Active Directory > Botón derecho > Nuevo > Usuario > Nombre > Contraseña
![Herramientas](../../assests/images/aso/ud6/act2/herramientas_usuarios_equipos.png)
![Nuevo usuario 1](../../assests/images/aso/ud6/act2/nuevo_usuario.png)
![Nuevo usuario 2](../../assests/images/aso/ud6/act2/nuevo_usuario_2.png)
![Nuevo usuario 3](../../assests/images/aso/ud6/act2/nuevo_usuario_3.png)
![Nuevo usuario 4](../../assests/images/aso/ud6/act2/nuevo_usuario_4.png)

NOTA: Se puede mover un usuario ya creado con el botón derechero sobre él y "mover".
![Mover usuario](../../assests/images/aso/ud6/act2/propiedades_usuario.png)

## Modificar un usuario
En las propiedades de una cuenta de usuario podemos realizar multitud de modificaciones:
- Opciones de contraseña
![Caducidad contraseña](../../assests/images/aso/ud6/act2/caducidad_password.png)
- Cambio de horario
![Horas de inicio de sesión](../../assests/images/aso/ud6/act2/horas_inicio_sesion.png)
- Miembro de un grupo
![Miembro de un grupo](../../assests/images/aso/ud6/act2/miembro_grupo_usuario.png)
![Miembro de un grupo 2](../../assests/images/aso/ud6/act2/miembro_grupo_usuario_2.png)

## Reestablecer la contraseña de un usuario
Para reestablecer la contraseña de un usuario: Seleccionamos el usuario > Acción > Reestablecer contraseña
![Reestablecer contraseña 1](../../assests/images/aso/ud6/act2/reestablecer_password.png)
![Reestablecer contraseña 2](../../assests/images/aso/ud6/act2/reestablecer_password_2.png)

## Crear copia de usuario
Botón derecho sobre el usuario > Copiar
![Herramientas](../../assests/images/aso/ud6/act2/herramientas_usuarios_equipos.png)
![Copia usuario 1](../../assests/images/aso/ud6/act2/copia_usuario.png)
![Copia usuario 2](../../assests/images/aso/ud6/act2/copia_usuario_2.png)

## Crear un equipo y modificarlo
Dentro de "Computers" > Botón derecho > Nuevo > Equipo
![Crear un equipo 1](../../assests/images/aso/ud6/act2/crear_equipo.png)
![Crear un equipo 2](../../assests/images/aso/ud6/act2/crear_equipo_2.png)
![Crear un equipo 3](../../assests/images/aso/ud6/act2/crear_equipo_3.png)

## Restricciones de usuarios y equipos
Botón derecho sobre el usuario > Cuenta > Iniciar sesión en
![Iniciar sesión en](../../assests/images/aso/ud6/act2/restriccion_usuario_equipo.png)

## Crear un grupo y modificarlo
En Users (o donde se requiera) > Botón derecho > Nuevo > Grupo
![Creando grupo](../../assests/images/aso/ud6/act2/crear_grupo.png)
Con el grupo seleccionado > Botón derecho > Propiedades
![Creando grupo 2](../../assests/images/aso/ud6/act2/crear_grupo_2.png)
![Creando grupo 3](../../assests/images/aso/ud6/act2/crear_grupo_3.png)
